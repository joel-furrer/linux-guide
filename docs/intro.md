# Introduction

Welcome to the Linux Guide.

This guide provides a structured introduction to Linux, Unix concepts, the command line, system administration, package management, and essential development tools.

The goal is not only to explain commands, but also the concepts behind them. Understanding how Linux works makes it easier to learn new tools and solve problems independently.

:::note

Before learning commands, it is important to understand concepts such as files, processes, users, permissions, shells, and standard input/output. The first chapters focus on these fundamentals.

:::

## This guide is intended for

- Linux beginners
- Students and self-learners
- System administrators
- Software developers
- Anyone interested in Unix-like operating systems

No prior Linux experience is required.
