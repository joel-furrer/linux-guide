### Die wichtigsten Markdown-Befehle (Notion-Vergleich):
| Was? | Markdown | Entspricht in Notion |
| :--- | :--- | :--- |
| Überschrift 1 | `# Titel` | `/h1` |
| Überschrift 2 | `## Untertitel` | `/h2` |
| Fett | `**Text**` | `Strg + B` |
| Code-Block | \` \` \`bash (Inhalt) \` \` \` | `/code` |
| Infobox | `:::tip` (Inhalt) `:::` | `/callout` |
