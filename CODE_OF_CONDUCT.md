# Code of Conduct

## Our Pledge

We are committed to making participation in this project a respectful and
welcoming experience for everyone.

## Expected Behavior

- Be respectful.
- Be constructive.
- Accept feedback professionally.
- Focus on improving the documentation.

## Unacceptable Behavior

- Harassment
- Personal attacks
- Discrimination
- Trolling
- Spam

## Enforcement

Project maintainers may remove comments, issues, pull requests, or other
contributions that violate this Code of Conduct.

## Scope

This Code of Conduct applies to:

- GitHub Issues
- Pull Requests
- Discussions
- Documentation reviews

## Contact

Open an issue marked as confidential or contact the maintainers.
