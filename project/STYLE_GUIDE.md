# Documentation Style Guide

## Documentation must be written in English.

## Code Blocks

Always use:

```bash
ls -a
```

Never use:

```
ls -lah
```

## Overview

## Why It Matters

## Concept

## Examples

## Common Pitfalls ( Optional )


---

---

## Syntax
