# Admonitions

Always use the best-fitting docusaurus admonition in the documentation.

```
:::note
This is a note.
:::

:::tip
This is a tip.
:::

:::tip[Custom Title]
This is a tip with a custom title.
:::

:::info
Additional information.
:::

:::warning
Keep this in mind.
:::

:::danger
This could lead to problems.
:::
```
